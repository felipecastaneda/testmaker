# **App Name**: TestMaker

## Core Features:

- Document Upload & Parsing: Allow users to upload Word (.docx), Excel (.xlsx), and plain text (.txt) documents for AI content extraction and preparation.
- AI Multiple-Choice Question Generation: Utilize an AI tool to read the document content and automatically generate a set of multiple-choice questions with a correct answer for each.
- AI Distractor Generation: An AI tool will generate plausible, incorrect answer options (distractors) for every multiple-choice question to create diverse 'flavors' of tests.
- Test Review Interface: Provide a clear user interface to display the generated test, allowing for quick review and basic inline editing of questions and answers.
- Test Download/Export: Enable users to download the complete multiple-choice test in a basic, human-readable format, such as plain text or PDF.

## Style Guidelines:

- Primary color: A calm, professional teal (#2898BD) for focus and intelligence.
- Background color: A very light, subtle blue-grey (#F0F3F4) to ensure maximum readability and a clean aesthetic.
- Accent color: A fresh, bright mint green (#52E0BD) to highlight interactive elements and important information.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and highly readable design across all text elements.
- Utilize minimalist, line-art style icons for functions like upload, download, edit, and question types to maintain a clean and professional appearance.
- Employ a clean, structured, and responsive layout that prioritizes clear information hierarchy and an intuitive flow, especially for document content and test generation steps.
- Implement subtle, unobtrusive animations for states such as loading (e.g., during AI processing) and hover effects, providing visual feedback without distraction.
